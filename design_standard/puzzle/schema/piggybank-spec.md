# 피기뱅크 (Piggy Bank) BM 파라미터

> **출처**: 퍼즐 기획자 제공 (2026-03)
> **용도**: Stage 0 BM 도메인 파라미터 정의 예시

## 기능 개요
유저의 핵심 플레이 활동을 유료 재화 누적으로 연결하여, 시각적 보상 경험과 함께 매력적인 구매 기회를 제공하는 기능. 첫 결제 장벽을 낮추고 반복 순환을 통한 안정적인 수익 창출 목표.

## 핵심 메커니즘
1. 기능 해금: 일정 레벨 도달 시 활성화
2. 재화 누적: 레벨 클리어 시 유료 재화 자동 적립
3. 가득 참 상태: 평균 7~8판 → 누적 중단, 구매 유도
4. 구매 및 보상: 상품 구매 → 누적 재화 즉시 획득
5. 사이클 반복: 구매 후 초기화, 재누적 시작

## 파라미터 정의
| 파라미터 | 설명 | 예시값 | 필수/선택 |
|----------|------|--------|-----------|
| price | 판매 가격 | $2.99 | 필수 |
| dynamic_pricing | 유저별 가격 차등 | False | 필수 |
| value_ratio | 가격 대비 보상 가치 배율 | 2배 (최소 200%) | 필수 |
| reward_composition | 보상 구성 | 100% 유료 재화(코인) | 필수 |
| unlock_condition | 해금 조건 | 20레벨+ | 필수 |
| fill_cycle | 가득 차는 평균 플레이 횟수 | 7~8판 | 필수 |
| exposure_condition | 노출 조건 | 가득 찼을 때 팝업 | 필수 |
| popup_priority | 레벨 클리어 후 팝업 순위 | 6순위 | 필수 |
| visual_feedback | 누적 단계별 시각 효과 | 색상 변화, 이펙트 | 선택 |
| sound_feedback | 청각 피드백 | 코인 적립 사운드 | 선택 |

## 설계 원칙
- 높은 가치 보장: 판매 가격 대비 최소 200% 가치
- 빠른 순환 주기: 7~8판으로 성취감 + 반복 구매 유도
- 직관적 보상: 100% 유료 재화로 가치 인지 용이
