### 4.2 해결하지 못한 근본 문제
1. **Board cleared 달성 0회**
   - 모든 세션이 235-290 cells 범위에서 종료 (시작 ~280-300)
   - 최대 진전: **~50 cells 감소** (시작 280 → 최저 235)
   - 100% 클리어까지의 80% 간격을 메우지 못함

2. **AI_DB 학습 편향**
   - (660, 1500) 좌표가 times_worked 1위 → 매 세션 우선 선택
   - d 값이 -35 → -1로 수렴했지만 여전히 선택됨
   - shuffle 수정도 효과 제한적 (DB query 상위 pool 내에서 여전히 포함)

3. **게임 메커니즘 이해 부족**
   - Fingerprint는 색 분포만 캡처, 홀더 상태/돼지 큐 공간 정보 없음
   - 홀더가 특정 색으로 포화 시 어떤 탭이 새 색 돼지를 불러오는지 모름
   - "pig queue → board color match → consumed" 메카닉의 세밀한 모델 없음

4. **YOLO 분류의 품질**
   - 일부 세션의 "win" 감지는 광고/결과 팝업을 오분류한 가능성
   - 실제 Board cleared와 YOLO "win"이 일치하지 않음
