### 1. 게임 정체성

| 항목 | 관찰값 |

| **부제** | 없음 (확인 불가) |

---

### 2. 런치 화면 (01_launch.png / 02_main_menu.png)

두 스크린샷 모두 **로딩 화면**으로 확인. 차이점은 하단 텍스트뿐:
- `01_launch.png` → `"Loading..."` (점 3개)
- `02_main_menu.png` → `"Loading.."` (점 2개)

**시각 요소:**
- 상단 중앙: "CAR MATCH" 로고 (황금색/주황색 윤곽선, 흰색 텍스트)
- **등장 캐릭터:**
  - 빨간 차 (주인공급, 지면에 위치)
  - 주황/노란 차 (공중에 점프/비행 중)
- **Mystery Box (? 박스):** 갈색 나무상자에 "?" 마크, 화면 곳곳에 4~5개 분산 배치
- 배경: 파란 하늘, 흰 구름, 좌측 파란 건물, 우측 핑크 건물

---

### 3. Daily Task 팝업 (03_settings.png)

> **주의:** 파일명은 `03_settings.png`이나 실제 화면은 **Daily Task 팝업**임. 설정(Settings) 화면이 아님.

#### 3-1. 상단 HUD (팝업 뒤 배경에서 확인)

| UI 요소 | 값 |
|---------|-----|
| **프로필 아이콘** | 차량 아이콘 (좌상단) |
| **코인** | 2,153 (황금 코인 아이콘 + "+" 버튼) |
| **하트(목숨)** | 5개 / "Full" 표시 (빨간 하트 + "+" 버튼) |
| **진행 바** | 0/1 표시, 우측에 무한대(∞) 심볼 + "15m" 타이머 |

#### 3-2. Daily Task 팝업 구성

**팝업 테마:** 주황-노란 테두리, 흰 배경, 제목 배너 "Daily Task"

| # | 태스크명 | 진행도 | 보상 |
|---|---------|--------|------|
| 1 | Win 1 level | 0 / 1 | 코인 50개 |
| 2 | Win 5 levels | 0 / 5 | 코인 150개 |
| 3 | Win 8 levels | 0 / 8 | 코인 200개 |

**Bonus Task 섹션:**
- 조건: "Claim all tasks to unlock!" (일반 태스크 3개 모두 완료 시 해금)
- 진행도: 0 / 3, 자물쇠 아이콘으로 잠금 상태 표시

**타이머:** "Ends in: 22h 14m" (하단, 갱신까지 남은 시간)

**닫기 버튼:** 우상단 빨간 원형 X 버튼

#### 3-3. 하단 내비게이션 바

| 위치 | 아이콘 | 라벨 |
|------|--------|------|
| 좌1 | 트로피/상자 아이콘 | (라벨 불확실) |
| 중앙 | 집 모양 아이콘 | **Home** (현재 선택, 강조 표시) |
| 우1 | 곰/캐릭터 아이콘 | (라벨 불확실) |
| 우2 | 톱니바퀴(설정) 아이콘 | (라벨 불확실) |
