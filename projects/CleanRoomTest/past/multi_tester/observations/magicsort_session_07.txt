## Magic Sort 게임 - 세션 7 분석 (재시작 흐름)

---

### 1. 게임 정체성
| 항목 | 값 |
|------|----|
| 제목 | Magic Sort (확인 불가, 앱 아이콘/타이틀 미노출) |
| 장르 | 캐주얼 퍼즐 / 소팅 (sort) 게임 |
| 아트 스타일 | 카툰 3D. 마법사 고양이 캐릭터 테마. 따뜻한 보라/주황 색조의 실내 배경 |
| 언어 | 한국어 |

---

### 2. 스크린샷별 상태

#### `01_playing.png` — 메인 맵 화면 (재시작 전)

**상단 HUD**
| 요소 | 값 |
|------|----|
| 캐릭터 아이콘 | 왼쪽 상단, 마법사 고양이 얼굴 (원형, 금테두리) |
| 코인 | 🪙 1420 (황금 동전 아이콘 + 숫자) |
| 하트(라이프) | 빨간 하트 + **5** (숫자) + 파란 **"전체"** 버튼 |
| 설정 버튼 | ⚙️ 오른쪽 상단 |

**중앙 화면**
- 배경: 마법 실내 공간 (책장, 창문, 가마솥)
- 캐릭터 3마리 노출: 파란 고양이(왼쪽), 회색-주황 고양이(중앙), 주황 고양이(오른쪽)
- 가마솥에서 분홍색 슬라임 물질이 넘쳐흐르는 연출

**중앙 하단**
- 초록색 큰 버튼: **"레벨 18"** (현재 진입 가능한 레벨)

**하단 내비게이션 바 (5개 탭)**
| 순서 | 아이콘 | 기능(추정) |
|------|--------|-----------|
| 1 | 책 모양 (파란색) | 스토리/미션 |
| 2 | 트로피 | 랭킹/업적 |
| 3 | 집 | **홈** (현재 선택됨, 텍스트 "홈" 표시) |
| 4 | 가방/선물 | 아이템 상점 |
| 5 | 고양이 얼굴 | 캐릭터/프로필 |

#### `03_after.png` — "여정" 팝업 (재시작 후 또는 맵 진행도 화면)

**팝업 구조**
| 요소 | 설명 |
|------|------|
| 팝업 제목 | **"여정"** (상단 보라색 탭 배경에 흰 텍스트) |
| 닫기 버튼 | 빨간 원형 X 버튼 (우측 상단) |
| 배경 질감 | 갈색 코르크보드/양피지 느낌 |

**진행도 트랙 (세로형 맵)**
수직 선로(초록+황금 색)를 따라 체크포인트가 나열됨. 현재 위치 마커: **18** (황금 원형 배지)

| 체크포인트 | 레벨 범위 | 보상/상태 | 잠금 여부 |
|-----------|----------|----------|----------|
| 레벨 10 | - | 🪙 코인 (황금 동전 아이콘, 초록 체크) | **완료됨** |
| 레벨 15 | - | ❤️ 하트 (무한 아이콘, 초록 체크) | **완료됨** |
| 현재 위치 | **18** | 황금 배지 마커 | 현재 위치 |
| **빗자루 비행** | 레벨 20–39 | 자물쇠 이미지 (갈색 잠금) | **잠김** |
| 레벨 25 | - | 🔄 되돌리기 x1 (화살표 아이콘) | (상단에 노출, 잠김 구역 안) |
| 레벨 30 | - | ❤️ 무한 하트 1시간 | (상단에 노출, 잠김 구역 안) |

**구간 명칭**
- **"빗자루 비행"**: 레벨 20–39 구간의 챕터/스테이지명. 현재 잠겨있음(자물쇠 이미지)

### 3. 게임 메커니즘 (추론 가능한 범위)
| 항목 | 관찰 내용 |
|------|----------|
| 장르 확인 | "Magic Sort" → 색/종류별 소팅 퍼즐로 추정 (이미지에서 게임 보드 직접 노출 없음) |
| 레벨 구조 | 선형 레벨 진행 (10, 15, 18, 20-39...) |
| 챕터 구분 | 레벨 범위별 챕터명 존재 (예: "빗자루 비행" = 20–39) |

### 6. 부스터/특수 아이템
| 아이템 | 확인된 정보 |
|--------|-----------|
| 되돌리기 (🔄) | 레벨 25 보상으로 x1 제공 예정. 노란 화살표 원형 아이콘 |
| 무한 하트 (❤️∞) | 레벨 15 완료 보상, 레벨 30 보상(1시간) |
| "전체" 버튼 | 상단 하트 옆 파란 버튼. 하트 전체 충전 기능으로 추정 |

### 7. 불확실 항목
- `01`과 `02`가 동일 화면 — 일시정지 UI가 별도 팝업 없이 처리되는지, GIF 동일 프레임인지 불확실
- 실제 게임 플레이 보드(소팅 필드, 튜브/슬롯 구조)는 3장 모두에서 미노출
- "전체" 버튼의 정확한 기능(하트 전체 충전인지, 전체보기인지) 불확실
- 하단 내비게이션 탭 4번, 5번 아이콘 기능 추정값
