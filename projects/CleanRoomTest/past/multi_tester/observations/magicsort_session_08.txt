### 2. UI 구성

#### 로딩 화면 (`01_start.png`)
- 상단 중앙: **MAGIC SORT** 로고 (금색+마법 파티클 이펙트)
- 하단 중앙: "**로딩 중**" 텍스트 (한국어)
- 하단: 빌드 번호 및 해시값 표시
- 배경: 풀 화면 일러스트 (3D 캐릭터 + 배경 합성)

#### 여정(Journey) 팝업 (`02_many_taps.png`, `03_extended.png`)
- 팝업 제목: **"여정"** (상단 파란 배너, 흰색 텍스트)
- 우측 상단: 빨간 **X 버튼** (닫기)
- 배경 텍스처: 낡은 종이/코르크보드 느낌, 스케치 그림(약병, 모래시계, 뱀) 장식
- 진행 경로: 세로 방향, **금색+초록색 수직 라인**으로 표현
- 현재 진행 배지: 경로 중간에 **초록 배지 "18"** 표시 → 현재 레벨 18 진행 중(불확실)

### 3. 게임 메커니즘

#### 여정(Journey) 시스템 구조
세션 8에서는 실제 게임플레이 화면이 아닌 **여정 팝업**이 반복 등장함. 이는 8회 탭 상호작용이 여정 팝업 진입으로 이어진 것으로 보임.

| 항목 | 내용 |
|------|------|
| 진행 방식 | 세로 스크롤 마일스톤 맵 |
| 챕터 단위 | 레벨 범위로 묶인 챕터 구조 확인 |
| 잠금 시스템 | 특정 챕터는 레벨 조건 미달 시 **자물쇠 아이콘**으로 잠금 |
| 조작 | 세션 내 8+ 탭 발생 → 화면 변화 없음(동일 팝업 유지) |

---

### 4. 재화/점수

세션 8 스크린샷에서 **코인/젬/점수 직접 표시 없음**.
여정 팝업 내에서 **보상 아이콘**만 확인됨:

| 보상 아이콘 | 위치 | 상태 |
|------------|------|------|
| 하트 무한 + "1시간" 텍스트 | 레벨 30 우측 | 미획득(흰 배경) |
| 황금 순환 화살표 "x1" | 레벨 25 우측 | 미획득(흰 배경) |
| 하트 무한 + 초록 체크 | 레벨 15 우측 | **획득 완료** (초록 체크) |
| 코인 더미 + 초록 체크 | 레벨 10 우측 | **획득 완료** (초록 체크) |

### 5. 난이도 / 진행 구조

| 마일스톤 레벨 | 보상 종류 | 상태 |
|--------------|----------|------|
| 레벨 10 | 코인 | 완료 ✅ |
| 레벨 15 | 무한 하트 | 완료 ✅ |
| 현재 진행 | **18** (배지) | 진행 중 |
| 레벨 20~39 | 챕터: **빗자루 비행** | **잠김 🔒** |
| 레벨 25 | 순환 부스터 x1 | 미획득 |
| 레벨 30 | 무한 하트 1시간 | 미획득 |

- **챕터명**: "빗자루 비행" (레벨 20~39) → 자물쇠로 잠긴 상태
- 현재 플레이어 위치: 레벨 18 (챕터 잠금 진입 직전)

### 7. 특수 기능 / 기타 관찰사항

| 항목 | 내용 |
|------|------|
| 여정(Journey) 팝업 | 레벨 마일스톤 보상 및 챕터 진행 확인 화면 |
| 챕터 잠금 | "빗자루 비행" 챕터가 잠겨 있음 → 레벨 20 이상 도달 시 해제 예상 |
| 무한 하트 시스템 | 여정 보상으로 제공됨 → 별도 라이프(하트) 시스템 존재 확인 |
| 빌드 정보 노출 | 로딩 화면에 빌드 번호와 해시값 표시 (QA/개발 빌드 특성) |

### 8. 세션 8 특이사항

- **02와 03 스크린샷이 동일한 화면** (여정 팝업, 차이 없음)
- 8회 이상의 탭이 발생했음에도 화면 전환 없이 **여정 팝업에 머무름**
- 실제 게임플레이 화면(보드, 소트 메커니즘)은 이번 세션에서 **관찰 불가**
- 로딩 → 여정 팝업 진입 경로로만 구성된 세션
