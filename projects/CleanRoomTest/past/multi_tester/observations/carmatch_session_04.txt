## Car Match - 세션 4 분석 보고서

---

### 1. 게임 정체성

| 항목 | 내용 |
|------|------|
| **제목** | CAR MATCH |
| **부제** | 없음 (로딩 화면 기준) |
| **장르** | 퍼즐 매칭 (차량 매칭 기반) |
| **아트 스타일** | 3D 카툰 렌더링, 눈이 달린 의인화 자동차 (Pixar/Cars 무비 유사 스타일) |
| **배경 테마** | 브라질 리우데자네이루 (코파카바나 광장풍, 크리스투스 구원자상 원경에 보임) |

---

### 2. 화면별 UI 구성

#### 01_main.png — 로딩/스플래시 화면

- 중앙 상단: "CAR MATCH" 타이틀 (노란색 + 갈색 외곽선, 큰 Bold 폰트)
- 캐릭터 2종 표시:
  - **빨간 차** (전면, 큰 사이즈) — 화난/진지한 눈
  - **주황/노란 차** (공중 점프, 작은 사이즈) — 놀란 눈
- 주변 오브젝트: "?" 마크가 적힌 갈색 나무 상자 4~5개 부유
- 배경: 컬러풀한 마을 거리 (좌측 초록/파란 건물, 우측 분홍 탑)
- 하단 중앙: **"Loading..."** 흰색 텍스트 (로딩 중)

---

#### 02_settings.png — 메인 맵 화면 (설정 아이콘 탭 전/후)

**상단 HUD (좌 → 우)**

| 위치 | 요소 | 값 |
|------|------|----|
| 좌측 상단 | 플레이어 아바타 아이콘 | 보라/청록 동물 캐릭터 (불확실) |
| 아바타 우측 | 골드 코인 아이콘 + 수치 | **2153** |
| 코인 우측 | 하트(라이프) 아이콘 | **5+** / **Full** 표시 |
| 최우측 | 타이머 | **15m** |

**진행바 (상단)**
- 파란 진행바: **0 / 1** (현재 진행 0, 목표 1)
- 우측 끝: **∞ (무한대)** 아이콘
- 하단: 시계 아이콘 + **10h 13m** 남은 시간 표시

**좌측 이벤트 패널**
- 원형 뱃지 아이콘 (레이싱/챔피언십 이벤트): **x1** 수량 표시
- 하단 레이블: **"Finished"** (완료 상태)
- 그 아래: **"Join"** 버튼 (파란 텍스트)

**우측 타이머**
- 건물/랜드마크 옆에: **22h 13m** 타이머 (시계 아이콘 포함, 불확실 — 이벤트 리셋 타이머로 추정)

**중앙 맵 장면**
- 빨간 차 캐릭터가 광장(코블스톤 바닥) 중앙에 위치
- 리우 배경: 크리스투스 구원자상, 알록달록 건물들

**하단 CTA 버튼**
- 초록색 큰 버튼: **"Level 70"** (현재 플레이 가능 레벨)

**하단 내비게이션 바 (파란색, 5개 아이콘)**

| 순서 | 아이콘 | 레이블 |
|------|--------|--------|
| 1 (좌) | 오렌지 상자/상점 | (상점) |
| 2 | 트로피 | (리더보드) |
| 3 (중앙) | 집 아이콘 | **Home** (현재 선택됨) |
| 4 | 카메라/사진 | (불확실) |
| 5 (우) | 톱니바퀴 | **설정 (Settings)** |

---

#### 03_back.png — 설정에서 돌아온 화면

- **02_settings.png와 레이아웃/수치 동일** (설정 팝업이 열렸다 닫힌 결과로 판단)
- 모든 HUD 값 동일: 코인 **2153**, 하트 **Full**, 타이머 **15m**, **10h 13m**, **22h 13m**
- Level 버튼: **Level 70**
- 시각적 차이 없음 → 설정 팝업이 일시적으로 열렸다가 닫힌 상태로 추정

---

### 3. 게임 메커니즘 (관찰 가능한 것)

- 차량 매칭 기반 퍼즐 게임 (제목 및 캐릭터 구성으로 판단)
- 레벨 번호 체계 존재 (현재 **레벨 70**)
- "?" 박스가 로딩 화면에 등장 → 랜덤 보상 박스 메커닉 존재 가능성

| 골드 코인 | 금색 원형 | **2153** | 상단 좌측 |
| 라이프(하트) | 빨간 하트 | **5+ / Full** | 상단 중앙 |

|------|----|
| 현재 레벨 | **70** |
| 진행 목표 | **0 / 1** |
| 라이프 상태 | **Full (최대)** |
| 라이프 회복 타이머 | **15m** |

---

### 6. 부스터

- 로딩 화면 및 맵 화면에서 부스터 아이콘 **직접 확인 불가**
- 이벤트 패널에 **x1** 수량 표시 → 아이템/부스터 보유 가능성
- 게임 플레이 화면 미진입으로 부스터 종류/수량 **미확인**

---

### 7. 특수 기능 및 설정

| 기능 | 상태 |
|------|------|
| 설정 버튼 | 하단 우측 톱니바퀴 아이콘으로 존재 확인 |
| 설정 팝업 내용 | **미캡처** (팝업 내부 화면 없음) |
| 이벤트 시스템 | "Finished" 완료 이벤트 + **22h 13m** 리셋 타이머 |
| 이벤트 참여 | **"Join"** 버튼 |
| 진행 타이머 | **10h 13m** (이벤트/챌린지 시간 제한으로 추정) |

### 종합 메모

- **설정 화면 캡처 미포함**: `02_settings.png`와 `03_back.png`는 설정 팝업 열림/닫힘 전후로 추정되나, 실제 설정 팝업 내부 UI가 캡처되지 않음. 다음 세션에서 설정 팝업 내부(음악/효과음, 알림, 언어 등) 캡처 필요.
- 현재 진행 상태: **레벨 70**, 코인 **2153**, 라이프 **Full** — 중반 이후 단계로 판단.
- 브라질 리우 테마 맵이 현재 활성 배경으로, 스테이지별 세계 여행 테마 구조 가능성 있음.
