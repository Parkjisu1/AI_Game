### Streak Race 결과 팝업 (`02_many_taps.png` / `03_extended.png`)

> `02`와 `03`은 **완전히 동일한 화면** — 장시간 플레이 후 상태 변화 없음.

**팝업 구조**
| 영역 | 내용 |
|------|------|
| 상단 이미지 | 3대 자동차 레이스 일러스트 (빨강/노랑/파랑) |
| 이벤트 타이틀 | **STREAK RACE** (노란-주황 그라디언트 텍스트, 체커 플래그 2개) |
| 상태 배지 | **Finished** (보라색 원형 배지) |
| 내 순위 | **Rank #50** |
| 내 보상 | **Reward: —** (보상 없음) |
| 닫기 버튼 | 우상단 빨간 원형 X 버튼 |
| 하단 버튼 | 초록 **Continue** 버튼 |

## 3. 게임 메커니즘

- 메인 메커니즘: 자동차 매칭 퍼즐 (직접 플레이 화면은 이번 스크린샷에 없음)
- **Streak Race**: 일정 기간 동안 연속 플레이(스트릭)로 점수를 쌓아 순위를 경쟁하는 이벤트
  - 결과 지표로 **플래그 아이콘 + 숫자** 사용 (레이스 포인트로 추정)

## 7. 주요 관찰 사항 (세션 8 특이점)

1. **Streak Race 50위 종료** — 보상 기준에 미달, Reward `-`
2. `02_many_taps.png`와 `03_extended.png`가 **완전히 동일** — 장시간 탭 후에도 화면 상태 변화 없음 (팝업이 고정된 채 유지)
3. **하트 Full 상태** — 장시간 플레이에도 하트 소진 없음 (Full 유지)
4. **코인 2153** — 세션 전반에 걸쳐 중간 수준 보유
5. 이벤트 진행도 `0/1` — 이번 세션에서 이벤트 목표를 달성하지 못한 상태로 시작
