#### `01_main.png` — 로딩/스플래시 화면

**레이아웃:**
- 상단 중앙: `MAGIC SORT` 로고 (입체적 레터링, 금색+청록색 그라디언트)
- 중앙 전체: 캐릭터 일러스트 (슬라이딩 카트/마법 양탄자 탑승 씬)
- 화면 하단: 로딩 텍스트

**캐릭터 (3종 확인):**
| # | 외형 | 특징 |
|---|------|------|
| 1 | 청회색 고양이 | 안경 착용, 왼쪽 배치 |
| 2 | 갈색 곰/너구리 | 대형, 중앙 탑승, 팔 들어올림 (흥분/환호) |
| 3 | 주황색 다람쥐/고양이 | 소형, 오른쪽 탑승 |

**배경 요소:**
- 마법 성/탑 (좌측 후경)
- 초록 숲 지형
- 공중에 떠다니는 게임 타일 조각들 (보라, 주황, 파랑 — 게임 핵심 오브젝트 암시)

**로딩 정보 (하단):**
- 텍스트: `로딩 중...`
- 숫자: `117702/1246` (로딩 진행 인덱스 또는 에셋 카운트, 불확실)
- 해시: `0e9fe113e528adbc504bbaf56f1b403e` (빌드 ID 또는 세션 토큰)

#### `02_settings.png` — 메인 허브 화면 (홈)

> **참고:** 파일명은 settings이나 실제 화면은 메인 허브(홈)임. 설정 팝업은 미열람 상태이거나 닫힌 후 복귀 상태.

**상단 HUD (Top Bar):**

| 위치 | 요소 | 값 |
|------|------|-----|
| 좌측 | 캐릭터 초상화 (원형 아이콘) | 주황/갈색 동물 캐릭터 |
| 중앙-좌 | 코인 아이콘 + 수치 | **1420** |
| 중앙-우 | 초록 원형 버튼 + 숫자 | **+5** (라벨: `전체`) |
| 우측 | 설정 아이콘 (⚙ 기어) | 탭 가능한 버튼 |

**메인 영역 (배경 씬):**
- 실내 마법 공방 배경 (벽난로/서재 분위기, 따뜻한 조명)
- 캐릭터 3종 재등장 (로딩 화면과 동일)
- 중앙에 핑크/보라색 슬라임/마법 포션이 끓어오르는 가마솥 연출

**핵심 CTA 버튼:**
- 중앙 하단 대형 초록 버튼: **`레벨 18`**
- 현재 진행 레벨 = **18**

**하단 내비게이션 바 (5개 탭):**

| 순서 | 아이콘 | 라벨 | 활성 여부 |
|------|--------|------|----------|
| 1 | 책/앨범 모양 | (불확실) | 비활성 |
| 2 | 트로피 | (불확실) | 비활성 |
| 3 | 성/집 아이콘 | **홈** | **활성 (현재)** |
| 4 | 가방/상점 | (불확실) | 비활성 |
| 5 | 캐릭터/프로필 | (불확실) | 비활성 |

#### `03_back.png` — 설정 복귀 후 화면

- `02_settings.png`와 **완전히 동일한 화면**
- 코인: **1420** (변동 없음)
- 레벨: **18** (변동 없음)
- 설정 화면 진입 → 뒤로가기 → 홈으로 복귀한 상태로 판단
- 게임 진행 상태나 재화 변화 없음 확인

| 재화 | 현재 값 | 위치 | 아이콘 |
|------|---------|------|--------|
| **코인** | **1420** | 상단 중앙-좌 | 금색 동전 |

> `전체` 버튼의 기능: 에너지 전량 충전 또는 친구 전체에게 보내기 기능일 가능성 (불확실)

### 5. 설정 화면 내용

- 세션 4에서는 설정 팝업이 **실제로 캡처되지 않음**
- `02_settings.png`는 설정 진입 전 또는 후의 홈 화면으로 추정
- 설정 아이콘(⚙)의 존재는 확인되나 내부 옵션(음악, 진동, 언어 등)은 미확인

### 6. 특이사항 / 추가 관찰

- **언어**: 한국어 (로딩 텍스트 `로딩 중...`, 탭 라벨 `홈`, 버튼 `전체`)
- **로딩 해시**: 세션별 고유 ID로 보임 (`0e9fe113...`) — 테스트 추적 목적 추정
- `02`와 `03` 화면이 동일한 것으로 보아, 설정 화면 캡처가 누락되었거나 설정 진입 없이 바로 복귀한 세션으로 판단됨

### 7. 미확인 항목 (추가 세션 필요)

- 게임플레이 보드 구조 (슬롯 수, 타일 색상 종류, 저장소 수)

- 힌트/되돌리기 기능 유무

- 라이프 시스템 상세 (최대값, 회복 시간)
