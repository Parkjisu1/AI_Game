## Tap Shift - 세션 5 분석 보고서 (보드 가장자리 탭)

---

### 주의 사항
3개 이미지(`01_board.png`, `02_edges.png`, `03_result.png`) **모두 동일한 화면**을 캡처하고 있습니다. 픽셀 수준에서 차이가 없어 동일 프레임으로 판단됩니다.

---

### 1. 게임 정체성

| 항목 | 값 |
|------|----|
| 장르 | 모바일 퍼즐 (레벨 맵 기반) |
| 아트 스타일 | 카툰/캐주얼 - 둥근 버튼, 밝은 색상, 3D 느낌의 UI 요소 |
| 현재 화면 | 레벨 선택 맵 (월드맵 뷰) |

---

### 2. UI 구성

#### 상단 헤더 (네이비 블루 바)
| 위치 | 요소 | 값/상태 |
|------|------|---------|
| 좌측 | 골드 코인 아이콘 + "+" 뱃지 | **85** 코인 |
| 우측 | 쇼핑 카트 아이콘 | 상점 버튼 |
| 우측 끝 | 기어 아이콘 | 설정 버튼 |

#### 우측 플로팅 UI
| 요소 | 설명 |
|------|------|
| ADS 뱃지 | 원형, 빨간 배경 "ADS" + 파란 라벨 "No Ads!" - 광고 제거 구매 버튼 |
| Claim 버튼 | 선물 상자 3개(물음표 포함) + 리본 이미지, 초록 "Claim" 버튼 - 보상 수령 |

#### 하단 고정 버튼
- 대형 초록 둥근 버튼: **"CONTINUE"** (흰색 볼드 텍스트)

---

### 3. 레벨 맵 구조

세로형 선형 맵. 원형 노드들이 수직 라인으로 연결됨. 화면에 보이는 레벨:

| 레벨 번호 | 색상 | 특이 표시 | 상태 추정 |
|-----------|------|-----------|-----------|
| 17 | 회청색 (상단 잘림) | 없음 | 미해금 또는 완료 |
| 16 | 회청색 | 없음 | 미해금 또는 완료 |
| **15** | **핑크/레드** | **두개골(💀) 아이콘** 노드 상단 | **보스/특수 레벨** |
| **14** | **밝은 초록** | 없음 | **현재 플레이 가능 레벨** |

#### 레벨 상태 색상 코드
- **초록** → 현재 진행 가능 (active)
- **핑크+두개골** → 보스 레벨 또는 실패한 레벨 (불확실)
- **회청색** → 잠금 또는 이미 클리어한 레벨 (불확실)

---

### 4. 재화/점수 시스템

| 재화 | 아이콘 | 현재 값 | 위치 |
|------|--------|---------|------|
| 코인 | 골드 원형 별 문양 코인 + 초록 "+" 뱃지 | **85** | 상단 좌측 |

- 젬, 별점, 하트 등 다른 재화는 현재 화면에서 **미확인**

---

### 5. 난이도 / 진행 구조

- 현재 진입 가능 레벨: **레벨 14**
- 화면에 노출된 레벨 범위: **14~17+**
- 레벨 15는 두개골 표시 → 위험/보스 구간 암시
- 선형 1차원 진행 (분기 없음, 현재 화면 기준)

---

### 6. 특수 요소

| 요소 | 상세 |
|------|------|
| 두개골 아이콘 | 레벨 15 노드 상단에 배치 - 특수 위험 레벨 표시 |
| Claim 버튼 | 물음표 선물 박스 - 랜덤 보상 수령 기능 존재 |
| No Ads 버튼 | IAP(인앱결제) 광고 제거 기능 존재 |

### 7. 이슈 / 테스트 결과

**중요: 3장 이미지가 모두 동일**
- `01_board.png` = `02_edges.png` = `03_result.png` (완전히 동일한 캡처)
- 세션명 "보드 가장자리 탭" 테스트 의도와 달리, **실제 게임플레이 보드 화면이 캡처되지 않음**
- 레벨 맵 화면만 3회 캡처된 것으로 보임
- 게임 내 보드(퍼즐 플레이 화면)와 가장자리 탭 결과 화면은 **미수집 상태**
