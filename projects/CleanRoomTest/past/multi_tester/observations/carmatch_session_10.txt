## Car Match 게임 - 세션 10 분석 보고서

### 1. 게임 정체성

| 항목 | 내용 |
|------|------|
| 게임명 | Car Match (추정, 상단 로고 아이콘으로 확인) |
| 장르 | 캐주얼 퍼즐 / 매치 게임 (맵 허브 기반) |
| 아트 스타일 | 밝고 채도 높은 카툰 3D 스타일 |
| 배경 테마 | 브라질풍 도시 (배경에 **예수상 실루엣** 확인) |
| 주인공 | 눈 달린 빨간 소형 자동차 (의인화 캐릭터) |

### 2. 화면 구성 (3장 모두 동일 - 메인 허브 화면)

**세션 설명 "빠른 플레이 + 뒤로가기"**: Level 70 버튼 탭 → (팝업 발생) → 뒤로가기 복귀 흐름으로 추정. 3장 모두 사실상 동일한 허브 화면 상태.

#### 상단 HUD
| 위치 | 요소 | 값 |
|------|------|----|
| 좌상단 | 게임 로고/캐릭터 아이콘 | - |
| 상단 좌측 | 골드 코인 | **2153** |
| 상단 중앙 | 라이프/하트 | **5 / Full** (녹색 버블 아이콘) |
| 상단 우측 | 이벤트 진행바 | **0/1**, ∞ 심볼, **15m** 카운트다운 |

#### 중간 UI (좌측 플로팅 요소)
| 위치 | 요소 | 값 |
|------|------|----|
| 좌측 상단 플로팅 | 배수 배지 (보라/핑크) | **x1** |
| 좌측 중단 | 원형 배지 (체커 플래그 패턴) | **"Finished"** 표시 |
| 좌측 하단 | 원형 배지 | **"Join"** (리그/토너먼트 참가 버튼 추정) |
| 우측 중단 | 원형 녹색 배지 | **22h 11m** 카운트다운 (이벤트 종료까지) |
| 중단 별도 표시 | 타이머 | **10h 11m** (에너지 혹은 이벤트 리셋 타이머 추정) |

#### 중앙 영역
- 코블스톤(돌바닥) 광장 위의 빨간 자동차 캐릭터
- 배경: 컬러풀한 건물들 + 원거리 예수상

#### 메인 CTA 버튼
- **"Level 70"** — 대형 녹색 버튼, 화면 하단 중앙 배치

#### 하단 네비게이션 바 (5개 탭)
| 순서 | 아이콘 | 활성 여부 |
|------|--------|-----------|
| 1 | 상점/쇼핑백 (불확실) | 비활성 |
| 2 | 트로피 | 비활성 |
| 3 | 집(Home) | **활성 (선택됨)** |
| 4 | 카메라/앨범 (불확실) | 비활성 |
| 5 | 설정(기어) | 비활성 |

### 3. 게임 메커니즘 (허브 화면 기준)

- **진행 방식**: 맵 기반 레벨 선택 구조 (현재 최전선 레벨 = **Level 70**)
- **조작**: Level 버튼 탭으로 레벨 진입
- **라이프 시스템**: 하트 5개 / Full (라이프 완충 상태)
- **이벤트 구조**: 시간 제한 이벤트 2개 이상 동시 진행 중

---

### 4. 재화/점수 시스템

| 재화 | 현재 값 | 아이콘 위치 |
|------|---------|------------|
| 코인 (골드) | **2153** | 상단 좌측, 노란 동전 아이콘 |
| 라이프 | **5 (Full)** | 상단 중앙, 녹색 아이콘 |
| 이벤트 진행 | **0/1** | 상단 우측 바 |

### 5. 난이도 / 레벨 진행

| 항목 | 값 |
|------|-----|
| 현재 도달 레벨 | **70** |
| 레벨 선택 구조 | 단일 CTA 버튼 (현재 레벨만 표시) |
| 완료 콘텐츠 | "Finished" 배지 1개 확인 (특정 이벤트/챕터 완료) |

### 6. 이벤트 / 특수 기능

| 요소 | 설명 |
|------|------|
| x1 배수 배지 | 점수/보상 배율 현재 1배 (불확실) |
| "Finished" 뱃지 | 완료된 이벤트/챕터 존재 |
| "Join" 뱃지 | 참가 가능한 리그 또는 팀 이벤트 존재 |
| 22h 11m 타이머 | 이벤트 종료까지 남은 시간 |
| 10h 11m 타이머 | 별도 타이머 (라이프 리필 또는 이벤트 리셋 추정) |
| 15m 타이머 | 상단 이벤트 미션 리셋 타이머 (불확실) |
