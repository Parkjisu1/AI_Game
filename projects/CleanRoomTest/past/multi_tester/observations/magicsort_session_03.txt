## Magic Sort - 세션 3 스크린샷 분석 (다중 탭 플레이)

---

### 1. 게임 정체성

| 항목 | 내용 |
|------|------|
| 제목 | Magic Sort (매직 소트) |
| 장르 | 퍼즐 / 소팅 (Sorting Puzzle) |
| 아트 스타일 | 캐주얼 카툰, 마법사/고양이 테마, 따뜻한 파스텔 계열 |
| 배경 테마 | 마법 아틀리에 (고양이 캐릭터 3마리, 가마솥, 책장, 아치형 창문) |
| 언어 | 한국어 |

---

### 2. UI 구성

#### 01_start.png - 메인 화면 (홈)
- **상단 바 (좌→우):**
  - 고양이 캐릭터 아이콘 (프로필/아바타, 좌상단 원형)
  - 코인 아이콘 (금색) + 수치 **1420**
  - 초록색 버튼 (+5 표시, "전체" 텍스트) → 하트/라이프 관련 버튼으로 추정
  - 기어 아이콘 (설정, 우상단)
- **중앙 메인 영역:**
  - 고양이 캐릭터 3마리가 가마솥 주변 배치 (분홍/보라 슬라임 넘침 연출)
  - 배경: 아늑한 실내, 책장, 아치형 창문, 따뜻한 조명
- **중앙 하단:**
  - 대형 초록색 버튼 **"레벨 18"** → 현재 플레이 가능한 레벨 진입 버튼
- **하단 내비게이션 바 (5개 탭):**
  - 아이콘 1: 상자/보물 상자 (좌측)
  - 아이콘 2: 트로피 (랭킹/업적)
  - 아이콘 3: 홈 (중앙, 성/집 모양, 현재 선택됨)
  - 아이콘 4, 5: (불확실, 우측 2개)

---

#### 02_mid.png / 03_after_many.png - "여정" 팝업

- **팝업 제목:** "여정" (Journey) — 보라색 배너, 흰색 텍스트
- **닫기 버튼:** 빨간 원형 X 버튼 (우상단)
- **배경:** 갈색 양피지 질감 + 마법 관련 낙서 (포션 병, 모래시계, 수식 등)
- **레이아웃:** 수직 경로 (초록+노란 대나무 형태 트랙)

### 3. 게임 메커니즘 - 여정(진행도) 시스템

#### 마일스톤 구조 (아래 → 위 진행)

| 레벨 | 보상 아이콘 | 보상 내용 | 상태 |
|------|------------|-----------|------|
| 10 | 코인(금색) + 초록 체크 | 코인 보상 | **획득 완료** |
| 15 | 하트 무한대(♾) + 초록 체크 | 무한 하트 | **획득 완료** |
| 18 | 초록 배지 (현재 위치 마커) | — | **현재 위치** |
| 20~39 | 자물쇠 이미지 카드 | "빗자루 비행" 챕터 | **잠금 (미개방)** |
| 25 | 순환 화살표 아이콘 + x1 | 부스터 1개 (불확실) | 미획득 |
| 30 | 하트 무한대(♾) + "1시간" | 무한 하트 1시간 | 미획득 |

#### 잠금 콘텐츠
- **이름:** "빗자루 비행" (Broomstick Flight)
- **조건:** 레벨 20~39 달성
- **잠금 상태:** 자물쇠 아이콘 + 어두운 미리보기 이미지
- **의미:** 챕터 또는 특별 이벤트 구간으로 추정

---

### 4. 재화/점수 시스템

| 재화 | 현재 값 | 위치 | 아이콘 |
|------|---------|------|--------|
| 코인 | **1,420** | 상단 바 좌측 | 금색 코인 |
| 하트/라이프 | +5 (전체) 버튼 표시 | 상단 바 중앙 | 초록 버튼 |

- 코인 정확히 **1420** 확인
- "전체" 버튼은 하트를 모두 채우는 기능으로 추정 (불확실)

---

### 5. 난이도 / 진행 구조

| 항목 | 내용 |
|------|------|
| 현재 레벨 | **18** |
| 확인된 마일스톤 레벨 | 10, 15, 18(현재), 25, 30 |
| 잠금 챕터 구간 | 레벨 20~39 ("빗자루 비행") |
| 획득 완료 보상 | 레벨 10(코인), 레벨 15(무한 하트) |
| 미획득 보상 | 레벨 25(부스터 x1), 레벨 30(무한 하트 1시간) |

---

### 6. 세션 3 특이사항 (다중 탭)

- **02_mid vs 03_after_many 비교:**
  - 두 이미지가 거의 동일 — 레벨 진행 없음 (여전히 레벨 18)
  - 배경 낙서 패턴 미세 차이 (우측 수식 낙서 추가) → 배경 애니메이션 또는 랜덤 렌더링
  - 획득 완료된 보상(레벨 10, 15) 상태 변화 없음
  - "다중 탭 플레이 5+" 후에도 **레벨 진행이 없음** → 세션 중 레벨 클리어 미발생 또는 여정 팝업만 반복 탐색

### 7. 특수 기능 / 기타

| 항목 | 내용 |
