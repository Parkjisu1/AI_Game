## 2. 스크린샷별 상세 분석

### 스크린샷 01 — 로딩 화면 (`01_level_select.png`)

> **실제 내용:** 레벨 셀렉트가 아닌 **로딩/스플래시 화면**

| 항목 | 내용 |
|------|------|
| **배경** | 판타지 성/탑, 초록 나무들, 황금빛 하늘 |
| **캐릭터 (3종)** | ① 파란 고양이 (좌), ② 곰/너구리 계열 (중앙 대형), ③ 다람쥐/청설모 (우) — 함께 비행 중 |
| **이동 수단** | 마법 카펫 또는 썰매 위에 탑승 |
| **로딩 텍스트** | `로딩 중...` (화면 하단 중앙) |
| **디버그 정보** | `117702124.6` / `0e9fe113e528adbc504bbaf56f1b403e` (해시값 노출) |
| **UI 요소** | 로딩 중 상태 → 상호작용 요소 없음 |

| 항목 | 내용 |
|------|------|
