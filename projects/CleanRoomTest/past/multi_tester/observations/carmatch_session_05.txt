# Car Match - 세션 5 분석: 보드 가장자리 탭

---

## 1. 게임 정체성

| 항목 | 내용 |
|------|------|
| 게임 제목 | Car Match |
| 화면 유형 | 메인 월드맵 / 홈 화면 |
| 장르 | 퍼즐 매칭 (캐주얼) |
| 아트 스타일 | 밝고 화려한 3D 카툰풍. 배경은 남미풍 도시 (Cristo Redentor 동상 식별 가능 → 리우데자네이루 테마) |
| 마스코트 | 빨간 SUV형 자동차 캐릭터 (크고 둥근 눈, 귀여운 표정) |

## 2. UI 구성

### 상단 바
| 요소 | 값/설명 |
|------|---------|
| 캐릭터 아바타 | 좌측 최상단, 보라색 자동차 얼굴 아이콘 |
| 코인 | 금색 코인 아이콘 + **2153** |
| 하트(라이프) | 초록색 하트 아이콘 + **5** + **"Full"** 표시 |
| 우측 타이머 | 무한대(∞) 기호 + **"15m"** (부스터/이벤트 잔여 시간으로 추정) |

### 중단 영역
| 요소 | 값/설명 |
|------|---------|
| 진행 바 | 보라/파란색 바, **0/1** 표시 |
| 진행 바 하단 타이머 | 시계 아이콘 + **10h 13m** (01\_board) / **10h 12m** (02,03) |
| 좌측 패널 1 | 회전 룰렛/스핀 아이콘, **×1** 뱃지, **"Finished"** 라벨 → 완료된 이벤트 |
| 좌측 패널 2 | **"Join"** 버튼 → 클랜/팀 참가 기능 |
| 우측 건물 | 초록 체크마크 아이콘 포함 건물, **22h 13m** 타이머 → 건설/이벤트 카운트다운 |

### 하단 영역
| 요소 | 설명 |
|------|------|
| "Level 70" 버튼 | 크고 밝은 초록색 버튼, 중앙 배치 → 현재 도전 레벨 진입 CTA |

### 하단 내비게이션 바 (5개 탭)
| 위치 | 아이콘 | 기능 |
|------|--------|------|
| 1번 (좌) | 선물/쇼핑백 아이콘 | 상점(Shop) |
| 2번 | 트로피 아이콘 | 리더보드/랭킹 |
| 3번 (중앙) | 집 아이콘 (파란색 강조) | 홈 (현재 활성) |
| 4번 | 카메라/앨범 아이콘 | 갤러리/스크린샷 기능 (불확실) |
| 5번 (우) | 톱니바퀴 아이콘 | 설정(Settings) |

## 3. 게임 메커니즘

- 세 스크린샷 모두 **인게임 플레이 화면이 아닌 월드맵/홈 화면**
- 실제 보드 구조는 이 세션에서 노출되지 않음
- "Level 70" 버튼이 현재 진입 가능한 레벨임을 시사
- 진행 바(0/1)는 현재 챕터 내 클리어 진도로 추정

| 기능 | 상태 | 비고 |
|------|------|------|
