### 3. UI 구성

#### 상단 HUD (좌→우)

| 위치 | 요소 | 값/설명 |
|------|------|---------|
| 최상단 좌 | 플레이어 아바타 아이콘 | 보라/핑크 고양이류 캐릭터 |
| 상단 좌 | 코인 아이콘 + 수치 | 🟡 **2153** |
| 상단 중 | 하트(라이프) | **5+** / **Full** 표시 |
| 상단 중-우 | 진행 바 (보라색) | **0/1** |
| 진행 바 우측 | 무한대 심볼 + 타이머 | ∞ / **15m** |
| 진행 바 하단 | 시계 아이콘 + 타이머 | ⏰ **10h 11m** |
| 우측 중단 | 타이머 (별도) | **22h 11m** |

#### 중앙 배경

| 항목 | 내용 |
|------|------|
| 배경 테마 | **리우데자네이루** 스타일 (브라질) |
| 랜드마크 | 원경에 예수상(Cristo Redentor) 실루엣 확인 |
| 건축물 | 컬러풀한 파스텔 톤 건물군 (파벨라/해안가 스타일) |
| 주인공 차량 | **빨간 차** 중앙 배치 (눈 있는 의인화, 정면) |

#### 좌측 이벤트 패널

| 항목 | 내용 |
|------|------|
| 배지 종류 | 체크무늬 깃발 뱃지 |
| 상태 표시 | **"Finished"** (완료) |
| 이벤트 버튼 | **"Join"** (다른 이벤트 참여 유도) |
| 배지 좌측 | **x1** 수치 표시 (배수/멀티플라이어 추정) |

#### 메인 액션 버튼

| 항목 | 내용 |
|------|------|
| 버튼 텍스트 | **"Level 70"** |
| 버튼 색상 | 초록색, 대형, 하단 중앙 |
| 의미 | 현재 플레이 가능한 레벨 = **70** |

#### 하단 네비게이션 바 (좌→우, 5개)

| 위치 | 아이콘 | 기능 추정 |
|------|--------|-----------|
| 1번 | 쇼핑백 | 상점 |
| 2번 | 트로피/컵 | 리더보드/업적 |
| 3번 | 집 (하이라이트) | **홈 (현재 선택)** — "Home" 텍스트 표시 |
| 4번 | 카메라/앨범 | 갤러리/이벤트(불확실) |
| 5번 | 톱니바퀴 | 설정 |

### 5. 난이도 / 레벨 구조

| 항목 | 내용 |
|------|------|
| 현재 레벨 | **70** |
| 레벨 표시 방식 | 허브 화면의 대형 그린 버튼 |
| 월드 테마 | 브라질/리우 (현재 진행 중인 월드로 추정) |
| 이전 이벤트 | Finished 상태 1개 확인 |
| 스크롤 구조 | 레벨 맵이 상하 스크롤 가능한 구조이나, 02/03에서 동일 위치 고정 확인 |

### 6. 타이머 분석

| 타이머 값 | 위치 | 기능 추정 |
|-----------|------|-----------|
| **15m** | 진행 바 우측 | 현재 이벤트 또는 라이프 충전 잔여 시간 (불확실) |
| **10h 11m** | 진행 바 하단 | 일일 보상 또는 이벤트 종료 카운트다운 (불확실) |
| **22h 11m** | 화면 우측 중단 | 별도 이벤트 또는 보상 잠금 해제 타이머 (불확실) |

---

### 7. 특이사항

- 로딩 화면(01)과 허브 화면(02/03) 사이 전환 — 세션이 게임 진입 직후 허브 화면까지를 캡처한 것으로 보임
- **02(scrolled_down)와 03(scrolled_up)이 동일**하게 보임 — 레벨 맵이 Level 70 이하로 더 스크롤되지 않거나, 스크롤이 같은 위치로 복귀한 것으로 추정
- 부스터, 힌트, 되돌리기 버튼은 이 스크린샷들에서 **확인 불가** (레벨 선택 화면이므로 인게임 UI 미표시)
- 장애물/터널/저장소 정보는 인게임 진입 후 확인 필요
