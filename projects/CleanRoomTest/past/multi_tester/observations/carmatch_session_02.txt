## Car Match - 세션 2 분석 보고서

### 1. 게임 정체성

| 항목 | 내용 |
|------|------|

### 2. UI 구성

#### 상단 HUD (메인 맵 화면)
| 위치 | 내용 |
|------|------|
| 좌측 상단 | 플레이어 아바타 아이콘 (보라색 프레임, 자동차 이미지) |
| 중앙 상단 | 코인 아이콘 + **2153** |
| 우측 상단 | 하트(라이프) **5** + **"Full"** 텍스트 |

#### 중간 진행 바
| 요소 | 내용 |
|------|------|
| 진행도 | **0/1** (키 수집 현황) |
| 좌측 뱃지 | **x1** (빨간/주황 배지 - 현재 Key Blaze 배율) |
| 우측 표시 | ∞ 아이콘 + **"15m"** (무한 라이프 15분 타이머, 불확실) |
| 하단 타이머 | 퍼즐 조각 아이콘 + **"10h 14m"** (이벤트 잔여 시간) |

#### 맵 노드 (좌측)
- 상단 노드: **"Finished"** (완료 표시, 체크마크/트로피)
- 하단 노드: **"Join"** 버튼 (이벤트 참가형)

#### 우측 건물
- 건물/상점 오브젝트 + 타이머 **"22h 14m"** 표시

#### 하단 내비게이션 바 (5개 탭)
| 순서 | 아이콘 | 기능 |
|------|--------|------|
| 1 | 분홍색 상점 | 스토어 |
| 2 | 노란 트로피 | 리더보드/랭킹 |
| 3 | 파란 집 | **Home** (현재 선택됨) |
| 4 | 카메라 | 소셜/캡처 |
| 5 | 톱니바퀴 | 설정 |

### 3. Key Blaze 팝업 (02_after_taps.png)

탭 2회 후 **Key Blaze** 팝업이 활성화됨.

#### 팝업 구조
- 제목: **"Key Blaze"** (보라 배경, 골드 테두리)
- 상단 좌: 파란 정보(i) 버튼
- 상단 우: 빨간 X 닫기 버튼
- 설명 문구: *"Beat levels and collect **keys** with the multiplier to earn more rewards!"*

#### 배율 탭 (상단)
| 탭 | 상태 |
|----|------|
| **x1** | 현재 선택 (주황 하이라이트) |
| x5 | 비활성 |
| x10 | 비활성 |
| x25 | 비활성 |
| x100 | 비활성 |

#### 보상 단계 (하단 → 상단 순)

| 단계 | 보상 아이콘 | 보상 내용 | 잠금 상태 |
|------|------------|-----------|-----------|
| **1** (현재 달성) | 무한대 기호 + 하트 | **무한 라이프 15m** | 해제됨 (빛나는 표시) |
| 2 | 코인 더미 | 코인 **50** | 잠김 (자물쇠) |
| 3 | 키 + x1 뱃지 | 키 부스터 x1 | 잠김 |
| 4 (최상단, 부분 노출) | 퍼즐 조각형 아이콘 | **100** (수량), 타이머 **10h 14m** | 잠김 |

### 4. 재화 / 점수 시스템

| 재화 | 값 | 위치 |
|------|----|------|

### 6. 특수 기능 및 이벤트

| 기능 | 내용 |
|------|------|
| Key Blaze | 레벨 클리어 시 키 수집 → 배율(x1~x100) 적용 → 단계별 보상 획득 |
| 이벤트 타이머 | 10h 14m 남음 (Key Blaze 종료 카운트다운) |
| 건물 타이머 | 우측 건물 22h 14m (별도 콘텐츠 잠금 해제 타이머) |
| Join 노드 | 멀티플레이어/협동 이벤트 참가 버튼 (불확실) |
| 무한 라이프 | Key Blaze 1단계 보상으로 15분 무한 라이프 활성화 상태 |
