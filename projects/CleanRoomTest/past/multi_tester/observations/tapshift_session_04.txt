## Tap Shift - 세션 4 분석 보고서

---

### 주의사항

**3장 모두 동일한 이미지**: `01_main.png`, `02_settings.png`, `03_back.png` 세 파일이 **완전히 동일한 화면**을 캡처하고 있습니다. 설정 화면(Settings)이 별도로 존재하지 않거나, 캡처가 동일 화면에서 3회 이루어진 것으로 보입니다. 분석은 공통으로 관찰되는 화면 기준으로 작성합니다.

---

### 1. 게임 정체성

| 항목 | 관찰값 |
|------|--------|
| 화면 유형 | 레벨 맵 (월드 맵 / 스테이지 선택) |
| 아트 스타일 | 캐주얼 모바일, 2D 플랫 + 약간의 3D 광택 효과 |
| 배경 색상 | 다크 네이비 (#0D1B2A 계열), 미세 도트 패턴 |

---

### 2. UI 구성

#### 상단 바 (Header)
| 위치 | 요소 | 값/설명 |
|------|------|---------|
| 좌측 | 골드 코인 아이콘 (별 문양, 노란색) + "+" 녹색 뱃지 | 코인 **85** 표시 |
| 우측 | 쇼핑카트 아이콘 (흰색 외곽선) | 상점 버튼 |
| 우측 | 기어(톱니바퀴) 아이콘 (흰색 외곽선) | 설정 버튼 |

#### 레벨 맵 (중앙, 수직 스크롤)
| 레벨 번호 | 색상 | 상태 추정 | 특이사항 |
|-----------|------|-----------|----------|
| 17 | 연회색/실버 | 잠금(미클리어) | 상단 부분만 노출(스크롤 위) |
| 16 | 연회색/실버 | 잠금(미클리어) | 일반 대기 상태 |
| 15 | **핑크/레드** | **현재 도전 레벨 or 실패** | 위에 **해골 아이콘** 표시 |
| 14 | **밝은 초록** | **클리어 완료** | 확인 가능한 최근 클리어 레벨 |

- 레벨 노드들은 세로 선(경로)으로 연결
- 노드 형태: 원형, 테두리(링) + 내부 색상 채움

#### 우측 플로팅 버튼
| 버튼 | 색상 | 설명 |
|------|------|------|
| ADS / **No Ads!** | 빨간 원 + 파란 탭 | 광고 제거 유료 상품 |
| **Claim** (선물 상자) | 녹색 버튼 + 선물박스 이미지 | 무료 보상 수령 버튼, 물음표(?) 선물 3개 표시 |

#### 하단
| 요소 | 색상 | 설명 |
|------|------|------|
| **CONTINUE** 버튼 | 밝은 초록, 둥근 직사각형 | 현재 레벨(15) 진입 버튼 |

---

### 3. 게임 메커니즘 (맵 화면에서 추정 가능한 수준)

- **레벨 기반 진행**: 번호 순서 진행 (아래→위 방향으로 올라가는 구조)
- **선형 맵 구조**: 분기 없이 일직선 경로
- **레벨 상태 색상 코드**:
  - 초록 = 클리어
  - 핑크/레드 + 해골 = 실패 또는 현재 도전 중
  - 회색 = 미해금

---

### 4. 재화/점수

| 재화 | 아이콘 | 현재 값 | 위치 |
|------|--------|---------|------|
| 코인 (골드) | 별 문양 노란 동전 | **85** | 상단 좌측 |
| 기타 재화(젬 등) | 미표시 | - | 이 화면에서 미노출 |

---

### 5. 난이도 / 진행 구조

- 확인 가능한 최소 레벨: **14** (클리어)
- 현재 도전 레벨: **15** (핑크+해골)
- 이후 레벨: **16, 17+** (잠금)
- 해골 아이콘: 레벨 15가 난이도 높음(위험) 또는 이전 실패 표시 가능성

---

### 6. 특수 기능

| 기능 | 확인 여부 |
|------|-----------|
| 힌트/부스터 | 이 화면에서 **미노출** (게임 플레이 화면에서 존재 가능) |
| 설정 | 기어 아이콘 존재, 클릭 후 화면은 **미캡처** |
| 광고 제거 구매 | "No Ads!" 버튼 존재 확인 |
| 무료 보상 | "Claim" 선물 박스 버튼 존재 확인 |
| 되돌리기/Undo | 이 화면에서 **미노출** |

### 7. 세션 4 특이사항 / 누락 정보

- **설정 화면 미캡처**: 세션 제목은 "설정 화면"이나 3장 모두 레벨 맵 화면으로 동일 → 설정 팝업이 열리지 않았거나 캡처 타이밍 문제로 추정
- **게임 플레이 화면 없음**: 실제 게임 보드, 조작 방식, 부스터 UI 확인 불가
