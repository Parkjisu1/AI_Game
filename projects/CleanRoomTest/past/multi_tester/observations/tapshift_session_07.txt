## Tap Shift 게임 - 세션 7 (재시작 흐름) 분석 보고서

---

### 주요 관찰: 세 스크린샷 모두 동일

`01_playing.png`, `02_pause.png`, `03_after.png` 세 파일이 **픽셀 단위로 동일한 이미지**입니다. 재시작 흐름 중 화면 전환이 발생하지 않았거나, 캡처 타이밍 문제로 동일 화면이 3회 기록된 것으로 보입니다.

---

### 1. 게임 정체성

| 항목 | 값 |
|------|----|
| 확인 가능한 화면 | 레벨 맵 (World Map / Level Select) |
| 장르 | 캐주얼 퍼즐 (추정) |
| 아트 스타일 | 단순한 원형 노드 맵, 다크 네이비 배경, 밝은 색상 UI |
| 게임 제목 | 스크린샷에 미표시 |

---

### 2. UI 구성

**상단 헤더 (네이비 바):**
- 좌측: 금색 코인 아이콘 (내부에 별 무늬) + 초록 "+" 뱃지 → 코인 수: **85**
- 우측: 쇼핑카트 아이콘 (상점), 기어 아이콘 (설정)

**우측 플로팅 UI:**
- **ADS / No Ads! 버튼**: 빨간 원형 배경에 "ADS" 텍스트, 하단 파란 레이블 "No Ads!" → 광고 제거 IAP 유도
- **Claim 버튼**: 색상 선물 박스 3개 (물음표 장식, 보우 리본) + 초록 "Claim" 버튼 → 보상 수령 기능

**하단:**
- **CONTINUE 버튼**: 대형 초록 둥근 직사각형, 흰 볼드 텍스트

---

### 3. 레벨 맵 구조

수직 체인 형태로 레벨 노드가 연결된 맵:

| 레벨 번호 | 색상/상태 | 특이사항 |
|-----------|-----------|----------|
| 17 | 회청색 (잠금) | 상단 부분만 보임 |
| 16 | 회청색 (잠금) | 미완료 상태 |
| **15** | **핑크/빨강 (현재 도전 대상)** | **노드 위에 핑크 해골(☠) 아이콘** → 하드/위험 레벨 표시 |
| **14** | **초록 (완료/현재 위치)** | 완료 또는 최근 클리어 |

- 노드 간 연결: 어두운 회색 수직 라인
- 배경: 다크 네이비, 미세한 도트 패턴

---

### 4. 재화/점수

| 재화 | 표시값 | 위치 |
|------|--------|------|
| 코인 (금색) | **85** | 상단 좌측 헤더 |

- 코인 아이콘: 금색 원 + 내부 별 무늬 + 초록 "+" 뱃지
- 기타 재화(젬, 별 등) 이 화면에서 미표시

---

### 5. 난이도 / 진행 구조

- 현재 진행 위치: 레벨 **14 완료**, 레벨 **15 도전 예정**
- 레벨 15에 **해골 아이콘** → 특수 난이도 표시 (하드 레벨 / 보스 레벨 추정)
- 레벨 16, 17은 잠금 상태 (회청색)
- 진행 방향: 하단 → 상단 (낮은 번호 → 높은 번호)

---

### 6. 부스터 / 특수 기능

이 화면(맵 화면)에서는 **부스터 UI 미표시**. 게임플레이 화면 진입 후 표시될 것으로 추정.
