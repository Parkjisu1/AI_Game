## 2. UI 구성

### 상단 헤더 (진한 네이비 바)
| 위치 | 요소 | 내용 |
|------|------|------|
| 좌측 | 코인 아이콘 | 골드색 동전(별 마크), 현재 값: **85** |
| 좌측 | "+" 버튼 | 코인 아이콘 하단, 초록색 원형 버튼 |
| 우측 | 장바구니 아이콘 | 상점(Shop) 진입 버튼 |
| 우측 | 톱니바퀴 아이콘 | 설정(Settings) 버튼 |

### 우측 플로팅 버튼 (2개)
| 버튼 | 설명 |
|------|------|
| **ADS / No Ads!** | 원형 뱃지, 빨간 배경에 "ADS" 흰 텍스트, 하단 파란 라벨 "No Ads!" → 광고 제거 IAP |
| **Claim** | 선물상자(노란/파란/보라 리본, "?" 마크 3개), 하단 초록 "Claim" 라벨 → 일일 보상 수령 |

### 하단
| 요소 | 내용 |
|------|------|
| **CONTINUE** 버튼 | 크고 둥근 초록색 버튼, 굵은 흰 텍스트 |

## 3. 레벨 맵 구조

### 노드 배치 (화면 기준 위→아래)
| 레벨 번호 | 색상 | 상태 | 특이사항 |
|-----------|------|------|----------|
| **17** | 회색/라이트블루 | 잠금(미완료) | 화면 상단에 부분적으로 잘림 |
| **16** | 회색/라이트블루 | 잠금(미완료) | 일반 노드 |
| **15** | 핑크/마젠타 | 특수(하드) | 노드 위에 **해골 아이콘** 부착 → 보스/하드 레벨 |
| **14** | **초록색** | 현재 선택/클리어 가능 | 가장 밝고 활성화된 색상 |

### 경로 연결
- 수직 파이프형 연결선 (진한 회청색, 3D 입체 효과)
- 노드 간 일직선으로 연결

### 노드 색상 코딩 (추정)
| 색상 | 의미 |
|------|------|
| 초록 | 현재 플레이 가능 레벨 |
| 핑크/빨강 + 해골 | 하드/보스 레벨 |
| 회색/라이트블루 | 아직 미개방 레벨 |

## 5. 난이도 / 진행 구조

- 확인 가능한 레벨 범위: **14 ~ 17** (화면 내 표시)
- 레벨 15에 해골 마크 → 특수 난이도 레벨 존재
- 레벨 번호는 **연속적 정수** 방식
- 진행 방향: 아래(낮은 번호) → 위(높은 번호)로 진행하는 수직 스크롤 맵
