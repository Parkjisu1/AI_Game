## Magic Sort - 전문가 9: 상태/흐름 매핑 분석

---

### 중요 관찰: 세션 캡처 이상

**6개 이미지가 모두 동일한 화면**을 보여주고 있습니다.

```
모든 스크린샷 = 동일한 Welcome/ToS 팝업 화면
배경: Magic Sort 로고 + 황금빛 배경
하단: "로딩 중..." (Loading...) 텍스트
세션 ID: 268435982
해시: 0e9fe113e528adbc604bbaf56f1b403e
```

---

### 1. 실제 캡처된 화면 분석

#### STATE_00: LOADING (로딩 화면)
```
표시 요소:
- Magic Sort 로고 (상단)
- "로딩 중..." 텍스트 (하단)
- 세션/빌드 ID 표시

레이어 위: WELCOME_POPUP (즉시 오버레이)
```

#### POPUP_00: WELCOME / ToS 동의 팝업
```
출현 조건: 최초 실행 또는 ToS 업데이트 시
내용:
- 제목: "Welcome"
- 본문: Terms of Service + Privacy Policy 동의 요청
       "If you are a returning player, please note
        that these have been recently updated."
- 버튼: [ACCEPT] (단일 버튼)
닫기 방법: ACCEPT 버튼만 가능 (강제 동의)
```

---

### 2. 파일명 기반 추론 상태 목록

파일명(`01_main_menu`, `02_gameplay`, ... `06_fail_popup`)으로 의도된 구조를 파악:

| 상태 ID | 파일명 | 화면 유형 | 실제 캡처 여부 |
|---------|--------|-----------|---------------|
| S0 | - | SPLASH / 로딩 | O (배경으로 확인) |
| S1 | 01_main_menu | MAIN_MENU | X (ToS 팝업으로 가로막힘) |
| S2 | 02_gameplay | GAMEPLAY | X |
| S3 | 03_pause_popup | PAUSE_POPUP | X |
| S4 | 04_complete_popup | COMPLETE_POPUP | X |
| S5 | 05_settings | SETTINGS | X |
| S6 | 06_fail_popup | FAIL_POPUP | X |

**결론: 세션 09는 ToS 팝업 단계에서 진행 불가 상태로 캡처됨.**

### 3. 전환 다이어그램 (확인된 부분)

```
[APP_START]
    |
    v

  "로딩 중..."

  - Terms of Service 표시
  - Privacy Policy 링크
    |
    | [ACCEPT 버튼 탭]
    v

    v

    |
    +--[일시정지 버튼]--> [S3: PAUSE_POPUP]
    |                          |
    |              +-----------+----------+
    |         [Resume]    [Restart]   [Home]
    |              |          |          |

                        [S1 복귀]
```

### 4. 팝업 목록 (추론 포함)

| 팝업 ID | 이름 | 출현 조건 | 확인 여부 |
|---------|------|-----------|-----------|
| P0 | Welcome / ToS | 최초 실행, ToS 업데이트 | **직접 확인** |
| P1 | Pause | 게임 중 일시정지 버튼 탭 | 파일명 확인 |
| P2 | Complete | 레벨 클리어 조건 달성 | 파일명 확인 |
| P3 | Fail | 실패 조건 (이동 소진 등) | 파일명 확인 |
