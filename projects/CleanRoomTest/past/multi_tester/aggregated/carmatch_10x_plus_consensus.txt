# Car Match — 전문가 10인 통합 합의 문서

---

## 공통 전제 (모든 섹션에 적용)

> **세션 전체 데이터 수집 실패 공지**
>
> 10개 세션(전문가 1~10) **전부** ToS 동의 팝업이 차단된 로딩 화면에서만 스크린샷이 캡처되었습니다. 게임플레이 화면은 단 한 장도 확보되지 않았습니다. 이하 문서는 **캡처된 로딩 화면에서 직접 측정된 값**과 **파일명·장르 관례에서 추론된 값**을 명확히 구분하여 작성합니다.

## 9. 타이밍 (Timing Expert — 전문가 4 담당)

> 인게임 타이밍 전량 미측정. **로딩 화면 한정** 데이터만 추출 가능.

| 항목 | 추정값 | 신뢰도 |
|------|--------|--------|
| ToS 팝업 등장 시간 | ~0.2–0.3s (Scale 0→1 + Fade-in 추정) | [LOW] |
| 팝업 등장 후 상태 | 정지 (애니메이션 없음) | [HIGH] |
| Loading 점 1사이클 | ~0.9–1.2s | [MED] |
| 프레임 간 캡처 간격 (추정) | ~0.3–0.5s | [MED] |
| 점 순환 패턴 | 0개 → 1개 → 2개 → 3개 → 0개 순환 (Linear) | [HIGH] |

**미측정 항목 (재캡처 필요):**
- 차량 이동 시작~완료 시간
- 이동 가속/감속 커브 (ease 방식)
- 이동 중 Stretch/회전 변형 여부
- 매칭 연출 시간
- 화면 전환 시간

```
APP_LAUNCH
    │ (자동)
    ▼
LOADING
    │ (리소스 완료)

TOS_POPUP ──[ACCEPT]──► MAIN_MENU ──[Settings]──► SETTINGS
(최초 1회)                   │                        │
                             │ [Play]                 │ [Close]
                             ▼                        ▼
                         GAMEPLAY ◄──────────── MAIN_MENU
                             │
              ┌──────────────┼──────────────┐
              │              │              │
         [Pause]        [성공 조건]    [실패 조건]
              │              │              │
              ▼              ▼              ▼
           PAUSED        COMPLETE        FAILED
              │              │              │
        [Resume/Home]  [Next/Home]    [Retry/Home]
```

## 재캡처 권장 사항 (공통)

모든 전문가가 동일하게 제안한 조치:

1. **ToS ACCEPT 버튼 클릭 처리 로직** 추가 (자동화 스크립트 수정)
2. **로딩 완료 대기 로직** 추가 (`Loading` 텍스트 소멸 후 캡처 시작)
3. 각 세션별 전문 화면 순서대로 재캡처
4. 재캡처 후 해당 전문가 세션 재분석 요청
