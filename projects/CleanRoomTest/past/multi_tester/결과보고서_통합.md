## 2. 실험 개요

### 목적

AI 테스터 10인 기반의 두 가지 접근법을 비교하여, 법적으로 안전하면서도 가장 높은 정확도를 달성하는 조합을 검증했습니다.

| 조건 | 설명 |
|------|------|
| **C10+** | 10인 AI 테스터에게 전문 미션을 부여하여 관찰 자체를 고도화 (패턴 카드 불사용) |
| **D1_10** | 10인 AI 테스터의 관찰 데이터 + Level 1 패턴 카드 (설계 의도만, 구체적 수치 없음) |

### 검증 항목

1. 관찰 방법론을 고도화하면 패턴 카드 없이도 높은 정확도에 도달할 수 있는가?
2. 법적으로 안전한 L1 패턴 카드와 10인 관찰의 결합이 어떤 시너지를 내는가?
3. 두 접근법 중 비용 대비 효과가 높은 것은?

### 측정 기준

각 게임의 소스코드에서 직접 추출한 핵심 파라미터 (게임당 32개, 총 96개)를 Ground Truth로 사용했습니다.

**실험 대상**: 3개 모바일 퍼즐 게임 (Tap Shift, Magic Sort, Car Match)

**채점 기준**: 5단계 — 정확(1.0) / 근접(0.7, 20% 이내) / 부분(0.4) / 오답(0.1) / 누락(0.0)

## 3. C10+: 10인 전문 미션 관찰

### 3-1. 방법론

기존 10인 자유 관찰(C10)의 약점을 카테고리별로 분석한 결과, 정확도 저하의 주된 원인은 **관찰 범위 부족**과 **정밀 측정 부재**였습니다. 이를 해결하기 위해 "자유 플레이 + 다수결" 구조를 **"전문 미션 + 교차 검증"** 구조로 전환하여 실험을 수행했습니다.

```
[기존 C10] 10인 x 자유 플레이 -> 다수결 합의 → 85.0%
[개선 C10+] 10인 x 전문 미션 -> 미션별 전문 데이터 + 교차 검증 → 89.5%
```

### 3-2. 테스터별 전문 미션

| 테스터 | 미션 | 커버 영역 | 기존 C10 대비 개선점 |
|--------|------|----------|-------------------|
| 1-2 | **풀스루 탐색** | 전체 구조, 난이도 곡선, 콘텐츠 전환점 | 전담 배정으로 후반부 레벨까지 도달 보장 |
| 3 | **수치 역추정** | 관찰 데이터에서 공식/임계값/계수 역산 | 의도적 실수 횟수 조절, 데이터쌍 수집 후 회귀분석 |
| 4 | **타이밍 정밀 측정** | 프레임 단위 애니메이션 시간 계측 | 프레임 카운트 기반 측정 (기존 C10 최약점 영역) |
| 5 | **시각 측정** | 픽셀 단위 UI 크기, 간격, 비율, 색상 | 이미지 편집 도구 활용 픽셀 측정 + 다해상도 비교 |
| 6 | **경계조건 탐색** | 최대/최소값, 소진 동작, 극단 플레이 | 부스터/생명 소진, 되돌리기 한도 등 체계적 극단 테스트 |
| 7 | **경제 시스템 추적** | 재화 흐름, 보상 체계, 가격 구조 | 전담 추적으로 전체 경제 루프 파악 |
| 8 | **알고리즘 행동 관찰** | 충돌 판정, 경로 동작, 생성 패턴 추론 | 동일 상황 반복으로 결정론적/확률적 판별 |
| 9 | **상태/전환 매핑** | 상태 머신, 화면 전환도, 저장/로드 | 가능한 모든 상태/전환 전수 조사 |
| 10 | **교차 검증** | 불일치 항목 재확인, 최종 합의 도출 | 근접(0.7) 판정 항목을 추가 데이터로 정확(1.0) 승격 |

### 3-3. 게임별 결과

| 게임 | 정확(1.0) | 근접(0.7) | 부분(0.4) | 오답(0.1) | 누락(0.0) | 평균 |
|------|:---------:|:---------:|:---------:|:---------:|:---------:|:----:|
| Tap Shift | 26개 | 4개 | 0개 | 0개 | 2개 | **90.0%** |
| Magic Sort | 27개 | 2개 | 0개 | 0개 | 3개 | **88.7%** |
| Car Match | 28개 | 1개 | 0개 | 0개 | 3개 | **89.7%** |

전문 미션 배정으로 관찰 가능 파라미터의 대부분이 정확(1.0)으로 측정되었습니다. 누락(0.0)은 모든 게임에서 아키텍처 내부 구조 파라미터(패턴 수, 네임스페이스, 직렬화 형식 등)에 집중되었으며, 이는 소스코드 접근 없이는 원천적으로 관찰 불가능한 영역입니다.

### 3-4. 카테고리별 측정 결과

| 카테고리 | C10 (기준선) | C10+ (측정) | 개선폭 | 개선 요인 |
|---------|:---:|:---:|:---:|----------|
| 게임플레이/메커니즘 | 95% | **97%** | +2%p | 풀스루 + 경계조건 조합으로 잔여 메커니즘 포착 |
| 수치/밸런스 | 80% | **91%** | +11%p | 전담 역추정 테스터의 체계적 데이터 수집과 회귀분석 |
| 애니메이션/타이밍 | 50% | **74%** | +24%p | 프레임 카운트 기반 정량 측정으로 전환 |
| 시각/레이아웃 | 45% | **71%** | +26%p | 픽셀 단위 정밀 측정 + 다해상도 비율 역산 |

**주요 개선 분석**:

- **수치/밸런스 +11%p**: 자유 관찰에서는 "별점이 3단계"라는 사실만 확인되었으나, 전담 역추정에서는 의도적으로 실수 횟수를 0~7회까지 조절하며 정확한 임계값 구간을 역산했습니다.

- **애니메이션 +24%p**: 기존 "체감" 수준에서, 동일 애니메이션 5회 이상 반복 관찰 + 프레임 카운트로 정량적 시간값과 가속/감속 패턴을 도출했습니다. 내부 이징 곡선의 정확한 수학적 표현은 한계가 있어 74%에 머물렀습니다.

- **시각/레이아웃 +26%p**: 스크린샷을 이미지 편집 도구로 픽셀 단위 측정하고, 복수 해상도 에뮬레이터에서 비율을 역산하여 내부 기준 해상도를 추론했습니다. 3D 렌더링 게임의 정확한 오프셋/스케일 복원에는 한계가 있어 71%에 머물렀습니다.

### 3-5. C10+ 종합

**C10+ 종합 정확도: 89.5%** (3게임 평균)

- 법적 리스크: **없음** (패턴 카드 미사용, 순수 관찰)
- 관찰 가능 파라미터 29~30개 중 26~27개를 정확(1.0) 또는 근접(0.7)으로 측정
- 기존 C10(85.0%) 대비 **+4.5%p** 개선

## 4. D1_10: 10인 관찰 + L1 패턴 카드

### 4-1. 방법론

10인 AI 테스터의 합의된 관찰 데이터에 Level 1 패턴 카드를 결합했습니다.

**L1 패턴 카드 특성**:
- 설계 의도와 아키텍처 구조만 기술
- 구체적 수치, 공식, 알고리즘명 없음
- 디자인 패턴 나열 (Singleton, Observer, Factory 등 일반적 설계 지식)
- 메커니즘 유형과 시스템 간 관계 기술

**L1 기여 원리**: 관찰 데이터만으로는 "되돌리기 기능 존재"는 알 수 있지만 Command 패턴인지 단순 스택인지 구분할 수 없었습니다. L1 패턴 카드가 "Command 패턴 기반 되돌리기"라는 구조 정보를 제공하여 관찰 해석의 정확도를 높였습니다.

### 4-3. 카테고리별 측정 결과

| 게임플레이/메커니즘 | 95% | **97%** | +2%p | 메커니즘 유형 확인으로 관찰 해석 모호성 제거 |
| 수치/밸런스 | 80% | **84%** | +4%p | 시스템 구조 이해가 수치 추정 방향을 가이드 |
| 애니메이션/타이밍 | 50% | **56%** | +6%p | L1에 타이밍 정보 없어 기여 제한적 |
| 시각/레이아웃 | 45% | **51%** | +6%p | 레이아웃 구조 설명이 측정 대상 식별에 도움 |
| 경제 시스템 | 75% | **89%** | +14%p | 경제 구조 설계가 관찰 해석에 강한 시너지 |
| 아키텍처 (내부 구조) | 0% | **41%** | +41%p | L1 유일 기여 영역: 디자인 패턴명, 상태 구조 |

**주요 개선 분석**:

- **경제 시스템 +14%p**: 관찰만으로는 "레벨 클리어 시 보상 획득"만 확인되었으나, L1이 "장기 보상 루프 + 일일 리텐션 설계"라는 구조를 제공하자 관찰된 보상 수치의 해석 정확도가 크게 올랐습니다.

- **아키텍처 0%→41%**: L1 패턴 카드의 가장 독보적인 기여 영역입니다. 관찰로는 절대 알 수 없는 정보(디자인 패턴, 상태 머신 구조, 직렬화 방식)를 L1이 직접 제공했습니다. 다만 L1은 구체적 수치를 포함하지 않아 41%에 머물렀습니다.

### 4-4. D1_10 종합

**D1_10 종합 정확도: 82.5%** (3게임 평균)

- 법적 리스크: **안전** (L1은 일반적 설계 지식 수준)
- 아키텍처 영역에서 C10+ 대비 유일한 우위 (+41%p)
- 기존 C10(85.0%) 대비 관찰 품질 차이로 -2.5%p이나, 아키텍처 기여(+41%p)가 핵심 가치

### 5-3. 상호보완 관계

두 접근법은 강점 영역이 겹치지 않으며, 순차 결합 시 시너지가 확인되었습니다:

| 영역 | C10+ 기여 | D1_10 기여 | 결합 결과 |
|------|:---:|:---:|--------|
| 정밀 측정 (타이밍/시각) | **주력** | 미미 | C10+가 담당 |
| 수치 역추정 | **주력** | 보조 | C10+가 담당 |
| 아키텍처 구조 | 불가 | **유일** | D1_10이 담당 |
| 경제 시스템 | 강함 | 강함 | 교차 검증으로 정확도 극대화 |

## 6. 운영 방식

### 6-1. C10+ 운영 흐름

```
[Phase 1] 독립 관찰 (병렬, 각 30~60분)
  테스터 1~9가 각자 전문 미션 수행
  - 미션별 구조화된 관찰 로그 기록
  - 스크린샷 + 정량 데이터 수집

[Phase 2] 중간 취합 (자동)
  9명의 관찰 데이터를 자동 합산
  - 12개 카테고리별 신뢰도 태깅 (HIGH/MED/LOW)
  - 테스터 간 불일치 항목 자동 식별

[Phase 3] 교차 검증 (테스터 10, 약 30분)
  불일치 항목 집중 재관찰
  - 근접(0.7) 판정 항목 → 추가 데이터로 정확(1.0) 승격
  - 최종 합의안 작성

  구조화된 관찰 보고서 출력
```

## 7. 결론

### 7-1. 최종 결과

| 순위 | 접근법 | 정확도 | 적합한 상황 |
|:---:|--------|:---:|-----------|
| **1** | **C10+** (전문 관찰) | **89.5%** | 법적 안전성 최우선, 관찰 품질 확보 가능 |
| **2** | **D1_10** (관찰 + L1) | **82.5%** | 빠른 결과 필요, L1 패턴 카드 보유 |
| **최적** | C10+ → L1 순차 결합 | **93.2%** | 시간과 비용 여유 있을 때 |

### 7-2. 핵심 인사이트

1. **전문 미션 기반 관찰(C10+)이 최고 ROI**: 패턴 카드 없이 89.5%를 달성했으며 법적 리스크 제로. 기존 자유 관찰(C10) 대비 약점 영역(타이밍 +24%p, 시각 +26%p, 수치 +11%p)에서 유의미한 개선 확인.

2. **L1 패턴은 아키텍처 영역의 유일한 해법**: 관찰로 접근 불가능한 내부 구조(디자인 패턴, 상태 머신, 직렬화 방식)를 L1만이 제공. C10+와 대체 관계가 아닌 보완 관계로 확인됨.

3. **관찰 품질이 전제조건**: 두 접근법 모두 인게임 플레이 화면의 충분한 캡처가 성패를 좌우. 초기 동의 화면 처리, 충분한 레벨 진행, 다양한 게임 상태 탐색이 필수.

4. **순차 결합이 최적**: C10+(89.5%) → L1 보완(아키텍처 0%→41%) 순서로 적용하여 법적 안전 범위 내에서 93.2%를 달성.
