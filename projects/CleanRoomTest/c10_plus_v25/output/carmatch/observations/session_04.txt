### 3. 실패 원인 분석

| 원인 | 영향 | 근거 |
|------|------|------|
| Windows 메모리 부족 | 전체 세션 실패 | WinError 1455 (paging file too small) |
| BlueStacks + Python + ADB 동시 | 리소스 고갈 | can't start new thread |
| 장시간 ADB 연결 | 세션 불안정 | 연속 screenshot failed |

### 4. 개선 방안

1. Windows 가상 메모리(페이지 파일) 크기 증설
2. BlueStacks 메모리 할당 감소 (2GB → 1.5GB)
3. 게임 교대 시 이전 게임 force-stop + 메모리 해제
4. ADB screencap 실패 시 ADB 서버 재시작 (adb kill-server && adb start-server)
