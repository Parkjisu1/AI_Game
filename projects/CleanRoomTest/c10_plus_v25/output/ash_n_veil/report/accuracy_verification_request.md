## 판정 기준

| 판정 | 점수 | 기준 |
|------|:----:|------|
| 정확 (Exact) | 1.0 | 값이 정확히 일치 |
| 근접 (Close) | 0.7 | 20% 이내 차이 또는 핵심 맞고 세부 다름 |
| 부분 (Partial) | 0.4 | 방향은 맞으나 수치 오차 큼 |
| 오답 (Wrong) | 0.1 | 완전히 다른 값 |
| 누락 (Missing) | 0.0 | AI가 미확인(null)으로 답한 항목 |

## 기입 후 처리

1. 위 표의 "실제값"과 "판정" 열을 기입
2. 또는 `ground_truth/ash_n_veil_ground_truth_template.yaml` 파일에 값 기입
3. 파일명을 `ash_n_veil_ground_truth.yaml`로 변경하여 같은 폴더에 저장
4. 파이프라인 재실행: `python run.py ash_n_veil --skip-capture`
5. 자동으로 정확도 점수 산출
