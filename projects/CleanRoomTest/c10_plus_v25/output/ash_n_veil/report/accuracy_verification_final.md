| **ADB 호환** | 단순 탭으로 게임 진행 가능 | 튜토리얼 가이드 따라가야 진행 |

## 7. 결론

### 7.1 정확도 25%의 의미

C10+ 방법론 자체의 문제가 아니라, **Idle RPG 장르의 구조적 특성**과 **ADB의 Unity 호환성 한계**가 결합된 결과입니다.

- 퍼즐 게임: 반복 구조 + 전체 UI 접근 → **89.5%**
- Idle RPG: 순차 해금 + UI 진입 실패 → **25.0%**

### 7.2 향후 개선 시 기대값

| 개선 | 해소 파라미터 | 정확도 변화 |
|------|:----------:|:---------:|
| UI 접근 문제 해결 (Replay/Human Capture) | Missing 15개 → Partial 이상 | 25% → **50~60%** |
| + 고레벨 데이터 추가 (Lv.30+) | Partial 12개 → Close 이상 | → **65~75%** |
| + 서버 밸런스 테이블 대조 | 서버 의존 18개 확정 | → **85~90%** |

### 7.3 애쉬앤베일 팀 요청 사항

현재 소스코드에서 추출한 Ground Truth에도 "server_data"로 표기된 항목이 18개입니다.
