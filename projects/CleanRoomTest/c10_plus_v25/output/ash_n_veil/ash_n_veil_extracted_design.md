## 2. 코어 루프

```
[자동 전투] → 경험치 / 골드 / 장비 획득
    ↓
[레벨업] → 스탯 상승 + 스킬 슬롯 해금
    ↓
[장비 강화] → 스크롤 소모로 장비 수치 상승
    ↓
[스테이지 진행] → 챕터별 맵 이동
    ↓
[가챠 소환] → 아바타 / 펫 / 완드 / 아티팩트 획득
    ↓
[반복] — 더 높은 스테이지로 진행
```

## 4. 캐릭터 성장 시스템 (Growth)

### 성장 공식 (코드 확정)

```csharp
// LevelUpChart.cs:42-55 — GetStat()
HP(Lv) = BasicStat[HP, key=1008] + SUM(LevelUp[1..Lv].stat_value where stat_key==1008)
ATK(Lv) = BasicStat[ATK, key=1000] + SUM(LevelUp[1..Lv].stat_value where stat_key==1000)
```

| 항목 | 구조 | 소스 |
|------|------|------|
| 기본 HP (Lv.1) | 서버 테이블 `BasicStat[stat_key=1008]` | `StatChart.cs:151` |
| 기본 ATK (Lv.1) | 서버 테이블 `BasicStat[stat_key=1000]` | `StatChart.cs:152` |
| HP 성장 | **누적 테이블** (선형 아님) — 레벨별 개별 증가분 합산 | `LevelUpChart.cs:42-55` |
| ATK 성장 | **누적 테이블** (선형 아님) — 동일 구조 | 동일 |
| EXP 곡선 | **레벨별 개별 테이블** (공식 아님) | `LevelUpInfo.exp` 필드 |
| 최종 스탯 | base + LevelUp 누적 + 장비 + 버프 | `StatChart.cs:209-237` |

### 데미지 공식 (코드 완전 추출)

```
CharacterPlayer.cs:650-720

damage = (baseATK + buffATK)             // 1단계: 기본 공격력 + 버프
       × (1 + criDamage)                 // 2단계: 크리티컬 (발동 시)
       × (1 + superCriDamage)            // 3단계: 슈퍼크리티컬 (발동 시)
       × (1 + finalAtkIncrease)          // 4단계: 최종 공격력 증가
       × (1 + weakness)                  // 5단계: 약점 보정
       - flatDamageReduction              // 6단계: 고정 피해 감소
       - percentDamageReduction            // 7단계: 비율 피해 감소
```
