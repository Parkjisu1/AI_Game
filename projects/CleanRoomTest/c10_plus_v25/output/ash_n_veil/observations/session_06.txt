## 2. 장비 등급 체계

현재 화면에서 확인된 등급 표시:

| 표시 | 의미 | 관찰 내용 |
|------|------|-----------|
| **III** | 등급 뱃지 (로마 숫자) | 모든 장착 장비가 `III` 등급 |
| **+0** | 강화 레벨 | 현재 강화 미진행 상태 |

> **추정 등급 체계**: 로마 숫자 방식 (I / II / III / IV / V ...) 으로 추정되나, 전체 최대 등급 확인을 위해 추가 스크린샷 필요

## 5. 추가 확인 요소

화면 하단 중앙의 두 아이콘:
- 🔥 형태 아이콘 (불꽃/화염) → 클래스/속성 선택 추정
- ❄️ 형태 아이콘 (눈결정) → 2번째 속성/전환 추정

## 6. 분석 한계 및 추가 필요 스크린샷

두 이미지 모두 **동일한 장비 개요 화면**만 보여주므로, 아래 항목은 **데이터 부재**:

| 미확인 항목 | 필요한 화면 |
|------------|-----------|
| 강화 비용 테이블 | 강화 팝업/메뉴 |
| 강화 최대 레벨 | 고강화 장비 or 강화 UI |
| 강화 성공률/실패 패널티 | 강화 상세 화면 |
| 전체 등급 색상 체계 | I~최대 등급 아이템 비교 |
| 세트 효과 | 세트 장비 장착 화면 |
| 장비 획득 경로 | 상점/가챠/드롭 화면 |
| 분해/매각 재화 | 가방 탭 > 장비 선택 팝업 |

## 결론 및 권장 추가 조사

현재 이미지로 확정된 정보:
- **슬롯 수**: 7개 (투구/무기/갑옷/신발/망토/화살통/장갑)
- **등급 표시**: 로마 숫자 `III` 방식
- **강화 레벨**: `+N` 형식, 현재 `+0`
- **핵심 스탯 5종**: 공격력/방어력/명중/HP/MP

완전한 장비 시스템 명세를 위해 **가방(Bag) 탭** 및 **장비 클릭 시 강화 팝업** 화면 추가 캡처를 권장합니다.
