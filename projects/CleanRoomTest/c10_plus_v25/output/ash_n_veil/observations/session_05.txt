### 9. 다해상도 비교 (단일 캡처 기준 역산)

이번 세션에는 이미지가 1장만 제공되어 해상도 간 직접 비교는 불가합니다.  
단일 캡처에서 **기준 해상도 역산**:

| 항목 | 계산 |
|------|------|
| 이미지 픽셀 크기 | 540 × 960 |
| 스킬 버튼 크기 | ~68 px |
| 1080p 환산 시 동일 버튼 | 68 × (1080/960) ≈ **76 px** |
| 1440p 환산 시 | 68 × (1440/960) ≈ **102 px** |
| Canvas Reference Resolution 추정 | **1080 × 1920 px** |
| Scale Factor | 0.5 (540/1080) |

> 추정 기준: CanvasScaler `Scale With Screen Size` → Reference `1080×1920`에서  
> 실제 기기 540×960으로 렌더링 → **Scale = 0.5**  
> 즉, 코드 내 설계 단위값 = 측정 px × 2
