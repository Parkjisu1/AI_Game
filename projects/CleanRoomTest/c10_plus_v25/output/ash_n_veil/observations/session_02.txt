## 1. 메인 HUD (게임 필드 화면 상단)

```
[상단 HUD]
├── 좌상단: 캐릭터 정보 블록
│   ├── 캐릭터 아바타 (원형 초상화, 탭 시 캐릭터 패널 열림 추정)
│   ├── 레벨 표시: "Lv.4" (step_1) / "Lv.2" (step_2 — 세션 차이)
│   ├── 닉네임: "Userdev306"
│   ├── HP 바 (적색, 상단)
│   └── 보조 스탯 수치 2줄 (EXP or 공격력/방어력 계열, "Xv / Yv" 형식)
│
├── 상단 중앙: 스테이지명
│   └── "그린밸" (Green Valley) — 현재 진행 필드명
│
├── 상단 우측: 재화 표시 (수평 배열)
│   ├── 파란 보석 아이콘: "0" (프리미엄 재화 — 다이아/젬)
│   └── 골드 코인 아이콘: "328,452" (step_1) / "183,095" (step_2)
│
└── 최우상단: 초록 사각형 버튼 (설정 또는 미니맵 추정)
```

## 2. 하단 네비게이션 바 (메인 탭 5개)

```
[하단 네비게이션 바]  — 좌→우 순서
│
├── [1] 방패/갑옷 아이콘  → 캐릭터/장비 패널
├── [2] 상자/가방 아이콘  → 인벤토리/가방 패널
├── [3] 중앙 원형 버튼    → 전투 재개 or 메인 필드 복귀 (주 액션 버튼)
├── [4] 활시위/궁수 아이콘 → 스킬/영웅 패널
└── [5] 자물쇠 아이콘     → 잠금 기능 (레벨 해금 조건 미충족 탭)
```

> **알림 dot 조건**: 각 탭 아이콘 우상단에 빨간 점 표시 조건:
> - [1] 캐릭터: 장비 업그레이드 가능 또는 레벨업 가능 시
> - [2] 인벤토리: 미확인 획득 아이템 존재 시
> - [4] 스킬/영웅: 스킬 포인트 or 영웅 강화 가능 시

## 3. 좌측 수직 단축 메뉴 바

```
[좌측 세로 버튼 (위→아래, 대부분 잠금 상태)]
│
├── 버튼 1: 자물쇠 (잠금) — 특정 레벨 해금 기능
├── 버튼 2: 자물쇠 (잠금)
├── 버튼 3: 자물쇠 (잠금)
├── 버튼 4: 체크리스트 아이콘 → 퀘스트/미션 목록 (추정)
├── 버튼 5: 자물쇠 (잠금)
├── 버튼 6: 자물쇠 (잠금)
└── 버튼 7: 자물쇠 (잠금)
```

> **패턴 분석**: 초반 세션(step_1, step_2 모두 저레벨)이라 대부분 잠금. 게임 진행에 따라 순차 해금되는 구조. 기능 후보: 소환, 상점, 업적, 길드, PvP, 이벤트 등.

## 4. 퀘스트/가이드 패널 (하단 중앙 플로팅)

```
[가이드 퀘스트 패널]
├── 아이콘 (화살표 > 버튼) — 패널 펼치기/접기

├── 진행도: "(10/10)" — 완료 상태
```

## 10. 화면 전체 트리 구조 요약

```
게임 필드 화면 (메인 루프)
│
├── 상단 HUD
│   ├── 캐릭터 정보 (아바타 / 레벨 / HP / 닉네임 / 스탯)
│   ├── 스테이지명 ("그린밸")
│   ├── 재화 (다이아 / 골드)
│   └── 설정 버튼 (우상단)
│
├── 좌측 수직 메뉴 (7개, 대부분 잠금)
│
├── 게임 필드 (3D 아이소메트릭)
│   ├── 마을/배경
│   ├── 게이트(아치문)
│   ├── 황금 소환석 (step_2 등장)
│   └── 자동 전투 캐릭터
│
├── 퀘스트 패널 (하단 플로팅)
│   ├── 가이드 퀘스트 텍스트
│   └── 진행도 + 보상 버튼
│
├── 채팅 티커 (서버 공지)
│
├── 하단 네비게이션 바 (5탭)
│   ├── [캐릭터/장비]
│   ├── [인벤토리]
│   ├── [메인/전투]
│   ├── [스킬/영웅]
│   └── [잠금]
│
└── EXP 바 (최하단)
```
